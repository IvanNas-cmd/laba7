# Лабораторная работа 7

## Description
Лабораторная работа №7. Разработка приложений для обработки графики, анимации и жестов на языке Swift. Вариант 17.

## Installation
Откройте проект в среде Xcode 14+ на macOS и соберите через `Cmd + B`.

## Usage
Приложение содержит экраны рисования (Objective-C), кастомной 2D графики с анимациями и жестами (Swift, MVC), а также 3D сцену на SceneKit.
